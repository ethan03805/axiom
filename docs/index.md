# Axiom Documentation

## Comprehensive Technical Reference

Axiom is an AI agent orchestration platform for autonomous software development. It coordinates multiple AI agents running in isolated Docker containers to build software from a single user prompt, with immutable scope, full audit trails, and fine-grained budget management.
